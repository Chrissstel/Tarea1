# Tarea1
implementación apropiada para las siguientes clases: STACK (lifo), QUEUE (fifo),
TABLE/HASH/DICTIONARY (order),..


TEST CASES DE STACK

TestStackPushPeek
- Insert elements into the stack.
- Verify that Peek() returns the last inserted element.

TestStackPop
- Insert multiple elements.
- Verify that Pop() returns the most recently inserted element (LIFO).

TestStackIsEmpty
- Verify that a new stack is empty.
- Verify that after inserting elements it is no longer empty.

TestPopEmptyStack
- Attempt to pop from an empty stack.
- Verify that the operation fails safely.
